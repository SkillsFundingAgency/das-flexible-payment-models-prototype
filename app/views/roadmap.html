Roadmap

This roadmap shows what we’re working on and planning to do.

Last updated 7 March 2023.

Recently shipped

We recently:

  • released new commitments journey for providers & employers
  • added functionality to capture the specific planned end date (previously we only captured month/year) to support more accurate on-programme earnings calculation
  • added a new indicative earnings report

Working on now

We're working to:

  • improve earnings calculations using specific planned end date
  • generate unfunded payments
  • prepare for the pilot launch
  • design and test a new change of price journey that would enable payments to be made from changes to Apprenticeship Service data

Exploring

We’re getting ready to:

  • start working on the technical design of flexi payments integration with v2 to support comparison testing (no production payments)
  • design and test a new change of start date journey to enable payments to be made from changes to Apprenticeship Service data
  • identify upstream (i.e Data Collections) and downstream reporting impacts from any new price change journey with suggested mitigations
